# C# Terminology
## Basics

### Methods & Classes are **Title Case** 
### Functions/Variable Names defined within "{}"
### .Write - leaves cursor at end of line
### .WriteLine - Moves cursor to next line
### .ReadLine - reads everything
### Comments defined by //
### All statements **must** end with ";"
### Variables and datatypes: *int, string, float, double, bool, long*
### Formatted Strings: `int.Parse number.ToString()`
